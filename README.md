## ✨ What this app does

same-energy-android is an Android app for image search with Same Energy, a visual search engine that finds images that match the look and feel of a photo.

Use it to:
- Search by image instead of words
- Find similar art, photos, wallpapers, and design ideas
- Browse results with a clean, modern layout
- Move fast through search results on your phone
- Keep the app simple to use

## 🖌️ App design

The app uses a Glassmorphism look. That means it uses soft transparency, smooth layers, and light blur effects. The result feels modern without making the screen hard to read.

The layout keeps the focus on images. Buttons and screens stay simple so you can search without extra steps.

## 🔐 Privacy and data

The app sends search requests to Same Energy so it can find similar images. It does not need a complex account setup for basic use. It keeps the flow simple and avoids extra sign-in steps.
